# Aimusic
## 这个项目是用google的人工智能框架tensorflow来自动生成曲谱的
## 此项目开源且遵循Apache协议，旨在用人工智能框架生成曲谱，帮助更多喜欢音乐的朋友创作分享自己的音乐

### 2018年4月2日晚开始，整体感觉还是有很大难度的，靠着年轻的心和一腔热血。
### 希项目目前刚起步望更多的感兴趣的朋友参与，更需要多的精神鼓励。
